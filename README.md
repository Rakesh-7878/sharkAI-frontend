# Shark AI

The Next Generation AI Model.

## Features
- **Responsive Design**: Professional UI that works on Mobile, Tablet, and Desktop.
- **Micro-animations**: Smooth transitions and interactive elements.
- **Free Offer System**: Interactive rewards and credit card modal simulation.
- **FastAPI Backend**: Real-time communication with the Shark AI engine.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Install backend dependencies: `pip install fastapi[all] fastapi-cors`
4. Run the backend engine (if applicable).
